2 · Deploy new hash

  1. Copy the hash above.
  2. Open auth.js on GitHub (opens in new tab).
  3. Find line 17 — PASSWORD_HASH: '...' — and replace the quoted value with your new hash.
  4. Scroll to the commit box, type Update access code, click the green Commit changes button.
  5. Wait ~30 seconds for GitHub Pages to rebuild, then reload command.azcustomknives.com and log in with the new code.
Heads-up: existing sessions stay logged in until their sessionStorage is cleared. Log out (close all CC tabs) to force a fresh login with the new code.
