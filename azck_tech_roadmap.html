The Big Picture

AZCK already uses Netlify (Command Centres), GitHub (source code), and Wix (websites). The gap is a real backend — a database, user accounts, real-time sync, and file storage. Supabase fills that gap and connects everything.

The Hunt Gallery project (tasks 31–37) already requires Supabase setup. That setup becomes the foundation for everything else. One Supabase project powers: Hunt Gallery, Command Centre upgrade, customer portal, inventory, and order tracking.

"Build the foundation once. Every feature after that is just a new table and a new page."

Phase 1 — Foundation NOW

Set up Supabase once. Everything else plugs into it. This phase costs $0 and unlocks Phases 2–4.

1A. Create the AZCK Supabase Project

One project for the entire business. Hunt Gallery, Command Centre, customer tools — all share this backend.

  1. Create project at supabase.com — name it azck-platform. Region: US East or Canada.
  2. Run the Hunt Gallery SQL setup (supabase-setup.sql from task #31) — this creates the first tables.
  3. Add Supabase keys to Hunt Gallery HTML files (task #32).
  4. Create admin accounts — Alessio (admin), Reanna (admin), Kyran (partner) — task #33.
  5. Add a tasks table for the Command Centre upgrade (replaces Gist sync).
  6. Add an orders table for future order tracking.
  7. Add a forge_scores table so Forge Evolution pulls live data.
Time estimate: 2–3 hours total
Cost: FREE 500MB database, 50K users, 1GB storage
Unblocks: Hunt Gallery (tasks 31–37), Command Centre upgrade, all Phase 2 projects

1B. Command Centre → Supabase

Swap GitHub Gist for Supabase. Same UI, better backend. Adds: real-time sync, dynamic categories (add/delete from the UI), and user-specific views (Alessio sees his tasks, Reanna sees hers).

  1. Create tasks and categories tables in Supabase.
  2. Update index.html to read/write from Supabase instead of Gist.
  3. Add "New Category" and "Delete Category" buttons to the UI.
  4. Add user authentication — Alessio and Reanna log in with their emails.
  5. Push to GitHub → auto-deploys to Netlify.
Time estimate: 3–4 hours
What changes for you: Categories become dynamic. No more redeploying to add/remove them. Tasks sync between devices instantly.

1C. Launch Hunt Gallery

This is already built. Once Supabase is set up (1A), it's just deployment.

  1. Add hunt-gallery.html to Shopify as Custom Liquid section (task #34).
  2. Host admin panel on Netlify or as hidden Shopify page (task #35).
  3. Test full flow — signup, upload, moderate, comment, share (task #36).
  4. Launch + announce on socials (task #37).
Time estimate: 2 hours (code is done)

Cost Breakdown

The entire Phase 1–2 roadmap runs on free tiers. The only paid upgrade worth considering early is Supabase Pro ($25/mo) to prevent the free tier's 1-week inactivity pause.

Service Free Tier Includes When to Upgrade Pro Cost
GitHub Unlimited private repos, 2,000 CI/CD min/month Likely never for AZCK's scale $4/user/mo
Netlify 100GB bandwidth, 300 build min, 125K function calls 10K+ monthly visitors $19/mo
