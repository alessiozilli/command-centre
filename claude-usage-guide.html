1. What Are Tokens?

Tokens are the units Claude uses to read and write text. Think of them like syllables — Claude doesn't read whole words, it breaks everything into small chunks called tokens.

Quick conversion

MeasureRoughly equals
1 token~4 characters or ~0.75 words
1,000 tokens~750 words (about 1.5 pages)
1 million tokens (1 MTok)~750,000 words (about 3 novels)

Real example: The forge.html file we just built is about 20KB of code. That's roughly 5,000 tokens. Every time Claude reads that file in a session, those 5,000 tokens count toward your usage.

2. What Costs You Tokens?

Every interaction with Claude has two sides — input (what Claude reads) and output (what Claude writes back). Both cost tokens. Output tokens cost 5x more than input.

Input tokens (what Claude reads)

WhatTypical cost
Your message50–500 tokens per message
Cowork system prompt (CLAUDE.md, skills, tools, safety rules)~15,000 tokens (carried every single turn)
Conversation historyGrows every turn — all previous messages are resent
File reads (Read tool)~250 tokens per KB of file
Web search results~2,500 tokens per page fetched
Images/screenshots~1,000–5,000 tokens depending on size

Output tokens (what Claude writes) — 5x more expensive

WhatTypical cost
A short reply~200–500 tokens
A detailed explanation~1,000–3,000 tokens
Writing a full HTML file (like forge.html)~5,000–15,000 tokens
A long report or document~10,000–30,000 tokens

The big one: Conversation history is the heaviest cost. Every time you send a message, Claude re-reads the entire conversation so far. A 50-message session means your last message carries all 49 previous exchanges. This is why long sessions use more tokens per message than short ones.

How a conversation grows

Each turn re-sends the full history. By turn 40, every message carries ~200K tokens of context.

7. Tips to Get More From Your Plan

Start fresh when context is heavy

When a session gets long (50+ turns), every new message re-sends the entire history. Starting a new session resets the context to zero. The Shared Brain means Forge picks up where you left off anyway.

Be specific up front

A clear first message ("update forge.html: cut outer ring spacing in half") costs fewer turns than a vague one that needs 3 rounds of clarification. Fewer turns = less context growth = fewer tokens.

Batch related tasks

Doing 5 small things in one session is cheaper than 5 separate sessions, because the system prompt (~15K tokens) loads once instead of five times.

Don't worry about it

Seriously. On Max 20x, you have massive headroom. The plan is designed for power users who work with Claude all day. Use it. That's what you're paying for.

8. The Bottom Line — What We Learned

After digging into the numbers, here's what actually matters:

What costs the most?

Long sessions. Not your messages. Not my thinking. Not even loading big files. The single biggest cost is conversation length — because every turn re-sends the entire history. Turn 1 costs ~$0.15 USD ($0.21 CAD). Turn 50 of the same session costs ~$1.50 USD ($2.10 CAD) for one message. The conversation gets 10x more expensive just by being long.

Does loading the Super Brain waste tokens?

No — it saves them. Loading the Super Brain costs ~$3.38 USD ($4.73 CAD) across a 30-turn session. But if Forge doesn't have that context and makes mistakes, each round of corrections costs ~$2+ USD ($2.80+ CAD) in re-sent context. Three corrections and you've already spent more than the Super Brain cost — plus you've lost 15 minutes of your time. Getting it right the first time is cheaper than saving on the load and paying in corrections.

Back-end instructions vs file reads — 7x cheaper

Knowledge in your CLAUDE.md (the back-end instructions) benefits from prompt caching. After the first turn, cached content costs $0.50 per million tokens instead of $5.00. That's 90% off. The same 22,567 tokens of Super Brain knowledge costs $3.38 USD ($4.73 CAD) in conversation context vs $0.47 USD ($0.66 CAD) in the system prompt over 30 turns — 7x cheaper. The plan: build a condensed "brain summary" into CLAUDE.md with the 20% of knowledge that matters 80% of the time.

The one habit that saves the most

Start fresh sessions sooner. When a conversation crosses ~30 turns and the work shifts to something new, say "save all" and start a new session. Forge picks up where you left off through the handoff file. A fresh session with full brain load = ~$0.15. Continuing a bloated old session = ~$1.50 per turn. This one habit would cut monthly usage by 20-30%.
